Welcome to the project!
This project is designed to spark your excitement for programming and help you dive deeper into learning. You'll be building a simple quiz game using HTML, CSS, and JavaScript. We've provided some starter code, and you'll update it step-by-step until you have a fully working game. No prior coding experience is needed — we'll explain what each part of the code does along the way. Get ready, have fun, and let's start coding!

Setup

In this section, you'll set up everything you need to get started.

  1. Download Visual Studio Code so we can edit the code. When the download is complete, open Visual Studio Code and install the Live server extension. This is used to view the project in the browser. For the rest of this tutorial, I will refer to Visual Studio Code as VSC.

  2. Download the starter code (animal-quiz.zip) to begin building your quiz game.
  3. Open the folder in VSC by either dragging it into the window or selecting it manually.

How to play the game

In this game, you will be asked a series of questions to guess the animal. Answer the questions.

Instructions:

  • Enter the correct animal name.
  • If your guess is correct, you go to the next question.
  • If not, you will get an alert that your answer is incorrect and a hint will be provided.
  • Complete all questions to win!
